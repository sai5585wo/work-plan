## 项目经历

### 跨境商品 Listing 半自动生成工作流（入门作品）

项目类型：个人项目 / 入门作品  
使用工具：飞书多维表格、AI 文案生成工具  
适用场景：商品资料整理、Listing 文案辅助生成、状态管理

项目简介：

搭建了一套基于飞书多维表格的跨境商品 Listing 半自动生成工作流。运营人员通过表单提交商品基础信息后，表格自动保存记录，并通过公式字段生成标准化 AI 提示词。AI 输出英文标题、五点描述、搜索关键词和风险提示后，再回填到表格中，并通过生成状态管理处理进度。

主要工作：

- 设计商品资料输入字段和 Listing 输出字段。
- 配置 `生成提示词` 公式字段，根据每行商品资料自动生成标准化 AI 提示词。
- 创建 `Workflow View`、`Demo View`、`Status View`、`To Generate View` 等视图。
- 创建商品资料提交表单，避免误填输出字段。
- 设置基础状态流程，管理待生成、已生成、需人工修改、已完成等进度。

项目定位：

该项目主要证明飞书多维表格基础能力、字段设计意识和流程管理意识，不作为核心求职竞争力。后续核心作品将转向 ERP/订单/库存/补货流程模拟。
