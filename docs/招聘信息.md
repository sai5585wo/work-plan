# 跨境电商 AI 自动化——关键信息与市场输入数据文档

> 本文档用于记录和归纳由用户（Yang Jianbo）提供的广州本地一线的真实市场用工需求、招聘卡点以及技术边界。通过对以下真实数据资产的逆向推导，作为项目全盘技术演进的唯一依据。

## 1. 核心用工样本数据

### 样本 A：广州南村（番禺区电商核心圈）卖家需求

*   **数据来源：** 广州如鱼得水文化信息科技有限公司——苏女士（招聘者）

*   **岗位/薪资：** AI自动化工程师（跨境电商TEMU） / 8-12K

*   **关键痛点输入（JD 抽离）：**

    1.  全链路自动化需求：涵盖选品、上架、客服、物流全全流程，急需减少人工重复点击。

    2.  中台协同：点名要求熟练开发“飞书多维表格/Airtable”，实现数据自动流转与清洗。

    3.  核心错配：老板用 8-12K 的薪资，试图招聘具备“机器学习、算法模型调优、全栈数据处理”的高级人才。

### 样本 B：广州嘉禾（白云区电商集群）卖家需求

*   **数据来源：** 广州远旸科技有限公司——吴女士（人事主管）

*   **岗位/薪资：** AI自动化工程师（跨境电商） / 10-15K

*   **关键痛点输入（JD 抽离）：**

    1.  数据孤岛问题：打通电商后台、ERP 系统（如店小秘、赛盒）、广告平台等离散数据，实现统一标准。

    2.  看板需求：实时监控销量、库存、广告 ROI、售后率等指标，减少人工统计。

    3.  AI 落地应用：基于 LLM API 开发浏览器插件，实现 Listing（标题、描述、关键词）多语言自动生成与评论情感分析。

## 2. 商业推论与交付边界数据

通过上述数据输入，反推得出我们在番禺及大广州地区做副业交付的三个“商业底线”：

1.  **别卖算法，卖表格：** 绝不向老板售卖复杂的代码或模型调优。老板和运营的痛点直观地反映在“飞书多维表格”的流转上，交付物必须以飞书表格看板形式呈现。

2.  **账号成本转嫁率 100%：** 影刀社区版 0 成本用于自身研发。交付时，优先导入甲方已有的企业账号（如样本 C 所示，跨境电商基本是影刀重度用户）；若无账号，则将 2499 元/年的专业版授权费作为硬性成本直接计入项目总报价中。

3.  **降维替代：** 我们的自动化方案旨在帮老板节省或替代掉原本需要花费 8-15K/月聘请的全职自动化研发人员，用一次性技术外包或年费维护的方式，让老板在极低的 ROI 风险下实现同等效果。
