# Mock Data（模拟数据）

这个目录用于存放跨境电商 ERP 流程模拟工作台的 CSV 模拟数据。

这些数据只用于学习、飞书多维表格导入和作品展示，不包含真实客户数据、真实订单数据、手机号、地址、账号密码或商业敏感信息。

## 文件对应关系

| CSV 文件 | 对应飞书表 | 说明 |
|---|---|---|
| `products.csv` | 商品资料表 | 商品 SKU、产品名、类目、平台、供应商、成本、商品状态 |
| `orders.csv` | 订单记录表 | 订单号、SKU、数量、销售额、平台、订单状态、订单日期 |
| `inventory.csv` | 库存表 | SKU、仓库、当前库存、安全库存、库存状态 |
| `replenishment.csv` | 补货采购表 | 缺口数量、补货原因、系统建议、补货状态、负责人、预计到货日期 |
| `exceptions.csv` | 异常记录表 | 缺货异常、订单异常、商品资料缺失等异常处理记录 |

## 建议导入顺序

建议按下面顺序导入飞书多维表格：

1. `products.csv` -> 商品资料表
2. `orders.csv` -> 订单记录表
3. `inventory.csv` -> 库存表
4. `replenishment.csv` -> 补货采购表
5. `exceptions.csv` -> 异常记录表

## 使用说明

- 文件名使用英文，便于 GitHub 管理。
- 表格内容使用中文，便于学习和展示。
- 所有表都围绕 `SKU` 做数据匹配。
- 导入飞书后，可以继续用 `SKU` 设置查找引用、公式字段和视图筛选。
